# Scripts — TruthBot Zero Hub

This folder contains executable scripts triggered by your Hub Bridge menu.

## Included
- `VaultOps_Cleanup_All.command` — Normalizes logs Aug–Dec 2025 and beyond, and seeds 2025-08-29 closing log.
- `Hub_Bridge.command` — TUI menu to run your common actions.

## Stubs
If `Open_Vault.command`, `Close_Vault.command`, `Sync_DryRun.command`, or `Deploy.command` are missing,
the Hub Bridge will auto-create stubs you can fill in.
